## Крафты

### Другие крафты

- Очищенная Золотая Пыль + Очищенная Кремнивая Пыль -> Золотая Полупроводниковая Смесь
- Очищенная Медная Пыль + Очищенная Кремнивая Пыль -> Медная Полупроводниковая Смесь
- Очищенная Алмазная Пыль + Очищенная Кремнивая Пыль -> Алмазная Полупроводниковая Смесь

- Урановая Пыль + Редстоун + Светокамень -> Ураново-Энергетическая Смесь

### Крафты в механизмах

- Кремний + **Дробилка** -> Дроблённый Кремний

- Дроблённый Кремний + вода + **Очеститель Руд** -> Очищенный Кремний
- Золотая пыль + вода + **Очеститель Руд** -> Очищенная Золотая Пыль
- Алмазная пыль + вода + **Очеститель Руд** -> Очищенная Алмазная Пыль
- Медная пыль + вода + **Очеститель Руд** -> Очищенная Медная Пыль

- Очищенный Кремний + **Сжиматель** -> Пластина Очищенного Кремния
- Алмазная Полупроводниковая Смесь + **Сжиматель** -> Процессорно-Полупроводниковая Кремнивая Пластина
- Золотая Полупроводниковая Смесь + **Сжиматель** -> RAM-Полупроводниковая Кремнивая Пластина
- Медная Полупроводниковая Смесь + **Сжиматель** -> SSD-Полупроводниковая Кремнивая Пластина
- Урановая-Энергетическая Смесь + **Сжиматель** -> Необработанный Ураново-Энергетический Слиток

- 2 Пластины Очищенного Кремния + **Полупроводниковый Производитель** -> Микроконтроллер
- Микроконтроллер + Процессорно-Полупроводниковая Кремнивая Пластина  + **Полупроводниковый Производитель** -> Процессор
- Микроконтроллер + RAM-Полупроводниковая Кремнивая Пластина  + **Полупроводниковый Производитель** -> RAM
- Микроконтроллер + Хранилище-Полупроводниковая Кремнивая Пластина  + **Полупроводниковый Производитель** -> SSD
