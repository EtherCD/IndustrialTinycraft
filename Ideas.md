Литеевое MFSU

Большее количество слотов для одновременной зарядки.
Возможность не взрываться от превышения номинала.
Молибденовый сплав в качестве корпуса

9 полимерных батарей = аккумуляторный массив

полимерная батарея = графит + стекловолокно + медь

4 пластин Молибденового сплава + 2 высокоэффективных провода + 2 аккумуляторных массивов + MSFU = Литиевое хранилище

взрывоустойчивый провод

Используется для создания фотонного процессора. Который уже используется в фотонном ускорителе.
